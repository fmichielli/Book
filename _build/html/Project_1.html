Part 2: Data Analysis without Pandas#

Read in data as a list of dictionaries and look at first values:

  • Split each row into columns with the delimeter “,”, since the datat is read in as a CSV

  • This will give individual values for each category (or column) per row

data = []
with open ("DHS_Daily_Report_20241028.csv", 'r') as DHS_data:
    keys = DHS_data.readline().rstrip('\n').split(",")
    data1 = DHS_data.readlines()

    for row in data1:
        row = row.rstrip('\n')
        rowlist = row.split(",")
        dict = {}
        for i in range(len(keys)):
            dict[keys[i]] = rowlist[i]
        data.append(dict)

Find the mean:

  • Sum the values and divide by a count of the values

sum = 0
count = 0
for n in adult_pop:
    sum += n
    count += 1
mean = round(sum/count,2)
print (f'The mean total adults in shelter is {mean}')

The mean total adults in shelter is 37981.03

Find the median:

  • sort the list

  • find the middle value or the two middle values of the list (and take the midpoint)

adult_pop.sort()
if len(adult_pop)%2 == 0:
    dist = len(adult_pop)/2
    median = (adult_pop[int(dist)] + adult_pop[int(dist-1)])/2
else: 
    median = adult_pop[int((len(adult_pop)-1)/2)]

print (f'The median total adults in shelter is {median}')

The median total adults in shelter is 36435.0

Find the mode:

  • Get set of unique values

  • Count the number of time each value occurs in the list

  • Report the value with the highest number of occurrences

  • If there are multiple modes this will report all

unique = set(adult_pop)
largest = 0
mode_set = {}
for n in unique:
    count = 0
    for i in adult_pop:
        if i == n:
            count += 1
    if count > largest:
        largest = count
        mode_set[n] = count
    if count == largest: 
        mode_set[n] = count
        
mode = []
for p in mode_set:
    if mode_set[p] == largest:
        mode.append(p)

print (f'The mode total adults in shelter is {mode}')

The mode total adults in shelter is [30715.0, 30943.0]
